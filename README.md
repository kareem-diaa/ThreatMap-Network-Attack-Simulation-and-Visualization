# ThreatMap-Network-Attack-Simulation-and-Visualization
# Project 25: ThreatMap – Network Attack Simulation & Visualization

## Scenario
A cybersecurity training institute wants a visual simulation to demonstrate how different attacks affect a network and how defenses respond.

---

## Functionality
- Simulate:
  - DoS
  - Port Scanning
  - MAC Flooding
- Display attack impacts on a visual network map

---

## Implementation
- Use **Cisco Packet Tracer** for network design and visualization
- Trigger:
  - Alerts
  - Log events for each attack

---

## Code Quality
- Organized CLI configurations with clear attack markers
- Clear ACLs and response rules

---

## Testing
- Validate the effect of each attack scenario
- Confirm defensive triggers:
  - Shutdown
  - ACL blocks

---

## Documentation
- Attack-response logs
- Visualization screenshots
- Network topology map
